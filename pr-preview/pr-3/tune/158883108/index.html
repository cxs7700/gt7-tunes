Unlike the 1.55 swap tune, which runs without ballast, this version uses 107 kg of ballast positioned at +50. The result is a slightly heavier car, but with 40 additional horsepower. This configuration makes it possible to run more rear downforce, reducing the PP enough to fit the extra power while also improving front-end precision. It's laping 6:30 Nordschleife and 2:17 on SPA To compare both versions, I ran ten laps of the Nordschleife using ghost replays. The ballast setup proved to be around one second per lap faster and, more importantly, felt better balanced and more rewarding to drive. The zero-ballast version tends to develop persistent mid-corner understeer. For those interested in making their own comparison, here is the zero-ballast tune. Enjoy your drive. ><(((((((°>°°°°°°
